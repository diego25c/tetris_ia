# Tetris IA
Juego del Tetris escrito enteramente por Bing Chat.

## Características
- Juego clásico de Tetris
- Creado enteramente por una IA

## Cómo jugar
Los controles aparecen en el propio juego:
* ← : Mover a la izquierda
* → : Mover a la derecha
* ↓ : Bajar más rápido
* ↑ : Rotar a la derecha
* [Barra espaciadora] : Bajar del todo
